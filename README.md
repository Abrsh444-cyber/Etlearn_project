# EthioLearn Pro 🎓

A bilingual (Amharic/English) AI-powered learning platform built for Ethiopian university students.

## Features

- 📚 **Exam Prep** — structured study materials and practice tests
- 🤖 **AI Tutoring** — personalized AI-powered academic support
- 🗂️ **Flashcards** — spaced repetition system for efficient memorization
- 🏛️ **University Directory** — information on Ethiopian universities
- 📊 **Analytics Dashboard** — track learning progress over time
- 🌐 **Bilingual Support** — full Amharic and English language support

## Tech Stack

- **Frontend:** Vite + TypeScript
- **Backend:** Node.js (server.ts)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

\`\`\`bash
git clone https://github.com/Abrsh444-cyber/Etlearn_project.git
cd Etlearn_project
npm install
\`\`\`

### Running locally

\`\`\`bash
npm run dev
\`\`\`

### Build for production

\`\`\`bash
npm run build
\`\`\`

## Deployment

This project is deployed via [Netlify](https://netlify.com).

- **Build command:** \`npm run build\`
- **Publish directory:** \`dist\`
